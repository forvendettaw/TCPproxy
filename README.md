# 目的

求人不如求己，为了更进一步了解python的socket的编程，加之我有一个想从外网控制内网树莓派的需求，我写了一个tcp代理程序。目前基本功能已经完成，还需要再打磨一下。

# 大致实现思路
非阻塞socket + 系统I/O复用机制(利用selectors模块)
